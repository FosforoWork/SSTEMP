### 🧩 Instrucciones Iniciales

1. **Archivo de Objetivos (GOALS.md)**: Define qué se quiere lograr, metas y cómo se medirán.
2. **Hoja de Ruta (ROADMAP.md)**: Divide el progreso en etapas semanales.
3. **Plan de Promoción (PLAN_DE_PROMOCION.md)**: Estrategia de marketing y redes sociales.
4. **Kit de Lanzamiento (LAUNCHKIT.md)**: Preparación para el lanzamiento.
5. **Flujos de Trabajo (WORKFLOWS.md)**: Descripción de procesos operativos.
6. **Agentes AI (AGENTS.md)**: Definición de agentes AI para el proyecto.
7. **Herramientas (TOOLS.md)**: Stack tecnológico seleccionado.
8. **Arquetipo (ARCHETYPE.md)**: Define rol, tono y audiencia.
9. **Entorno (ENVIRONMENT.md)**: Configuración del entorno de desarrollo.
